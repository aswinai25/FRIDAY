🚀 QUICK START - 3 COMMANDS TO GET LIVE:

1. PUSH TO GITHUB (Make it public)
   ─────────────────────────────────────
   git init
   git remote add origin https://github.com/YOUR_USERNAME/friday-ai.git
   git add .
   git commit -m "F.R.I.D.A.Y. AI Production Deployment"
   git push -u origin main

2. DEPLOY BACKEND (Choose Render recommended)
   ─────────────────────────────────────
   • Go to https://render.com
   • Sign up with GitHub
   • Connect your friday-ai repo
   • Use defaults (gunicorn will work!)
   • Add env vars (FLASK_ENV=production, etc.)
   • Deploy!
   • Copy your backend URL: https://friday-ai.onrender.com

3. DEPLOY FRONTEND + CONNECT
   ─────────────────────────────────────
   • Go to https://netlify.com
   • Connect your GitHub repo
   • Deploy! (takes 30 seconds)
   • Edit index.html: Replace "localhost:5000" with your backend URL
   • Git push
   • Done!

💡 PRO TIPS:

1. Test Locally First
   python app.py
   # Make sure it works before deploying

2. Use .env for Secrets
   • Never commit .env to GitHub
   • Use .env.example as template
   • Add real values in platform settings

3. Monitor Your App
   • Check Render/Netlify dashboards
   • Review deployment logs
   • Watch for errors

4. Share Wisely
   • Backend URL = internal only
   • Frontend URL = share this!
   • GitHub = show this to developers

5. Custom Domain Later
   • Don't need now (use free subdomain)
   • Add custom domain (~$12/yr) later
   • Both Render and Netlify support it

❓ FAQ - QUICK ANSWERS:

Q: Why do I need both backend & frontend?
A: Backend = AI brain (processes messages)
   Frontend = UI (what users see)
   They talk to each other via API

Q: Can I deploy just to one platform?
A: Both suggested (Render+Netlify) are free
   Together = best experience
   Single = limited functionality

Q: What if something breaks?
A: Logs are in platform dashboards
   Check DEPLOYMENT_GUIDE.md → Troubleshooting
   Most issues: restart service or check URLs

Q: How much does it cost?
A: $0/month with free tiers
   Optional: Custom domain (~$12/year)
   Upgrade to paid tiers only if needed

Q: Can people see my code?
A: Yes! It's public on GitHub
   That's the point of open source
   They can learn from it / fix bugs

Q: How do I update the AI later?
A: 1. Edit code locally
   2. Git push to GitHub
   3. Platform auto-deploys
   4. Done! (live in 1-2 minutes)
