TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

ISSUE: "Repository not found"
→ Make sure you have authorized Render/Netlify with GitHub
→ Go to GitHub settings → Applications → make sure Render/Netlify has access

ISSUE: "Build failed"
→ Check the build logs in Render dashboard
→ Usually means Python dependencies issue
→ You can ignore this first time - just click "Retry"

ISSUE: "Frontend can't connect to backend"
→ Make sure you updated index.html with correct backend URL
→ Make sure you git pushed the changes
→ Give Netlify 2 minutes to rebuild

ISSUE: "Service shows error"
→ Go to Render dashboard → Your service → Logs
→ Look for error messages
→ Usually solves itself - wait 5 min and refresh

ISSUE: "Slow on first load"
→ Render free tier sleeps after 30 min of inactivity
→ First request wakes it up (takes 10-15 sec)
→ This is normal and FREE!
